# CocoAi Study MVP Release Checklist

## 시나리오 1: 기본 학습 복원
1. 앱 실행
2. 이미지 또는 PDF 업로드
3. 문제 분석 확인
4. 정답/풀이 요약 확인
5. 앱 종료
6. 재실행
7. 마지막 파일/문제 복원 확인

## 시나리오 2: 프롬프트 중심 학습 흐름
1. `풀이`
2. `정답만`
3. `다시 설명`
4. `더 쉽게`
5. `왜 이렇게 돼?`
6. `선행`
7. `이전 문제`
8. `현재 문제 뭐야?`

## 시나리오 3: 오류 처리
1. 실패 이미지 업로드
2. 미지원 문제 입력
3. current problem 없이 follow-up 입력
4. 손상된 JSON 로드
5. 저장 실패 상황 재현

## 목표 기준
- 앱 시작: 3~8초
- 이미지 분석: 3~15초
- 저장된 문제 열기: 1초 이내
- 풀이/정답 응답: 1~3초
- 선행 생성: 2~5초

## 배포 전 판정
- MVP 배포 가능
- 추가 수정 필요
- 성능 최적화 필요
- 오류 처리 보강 필요
