# COCO Study Sample Level Diagnostic Flow

사용자:  
나는 초등학교 3학년부터 수학을 포기했어.

COCO:  
괜찮아요. 수학을 오래 놓았다고 해서 다시 시작 못 하는 건 아니에요. 좋아요, 우리 지금 실력을 재는 게 아니라 어디서부터 다시 연결하면 좋을지 같이 찾는 거예요.

부담 없이 짧게만 말해줘도 괜찮아요.

진단 질문: 사과 3개가 있고 2개를 더 받는다고 할 때, 머릿속에 어떤 장면이 떠올라?

사용자:  
그냥 5가 된다는 생각만 들어.

COCO:  
좋아요. 지금 답으로 방향이 조금 보여요.

부담 없이 짧게만 말해줘도 괜찮아요.

진단 질문: 사탕 8개를 2명이 똑같이 나눈다는 건 어떤 뜻으로 느껴져?

사용자:  
둘이 4개씩 가지는 거?

COCO:  
좋아요. 지금 답을 보면 현재 출발점은 대략 레벨 0 쪽으로 보여요.

특히 먼저 보강하면 좋은 부분은 수의 의미, 나눗셈의 의미예요.

추천 단원:
- 수의 의미
- 덧셈/뺄셈 의미
- 곱셈 의미
- 나눗셈 의미

우리 바로 '수의 의미'부터 같이 시작해보자.
